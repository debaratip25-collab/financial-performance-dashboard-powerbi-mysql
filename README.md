# Enterprise Financial Performance Dashboard | Power BI + MySQL

## Overview

This project is an end-to-end Business Intelligence solution designed to analyze enterprise financial performance using Power BI and MySQL.

The dashboard provides insights into revenue trends, profitability, regional performance, product category contribution, and strategic growth opportunities through interactive visualizations.

---

## Tech Stack

- Power BI
- MySQL
- SQL Views
- Python (for realistic synthetic data generation)
- DAX Measures

---

## Dataset Overview

The project uses a star-schema based relational model with realistic business logic and financial constraints.

### Fact Tables
- Fact Revenue
- Fact Expenses
- Fact Budget

### Dimension Tables
- Date
- Region
- Product
- Department
- Channel
- Segment

## Key Business Insights

- October recorded the highest revenue month, indicating strong Q4 demand.
- North region ranked #1 in revenue contribution.
- Furniture category generated the highest revenue and profit.
- East region underperformed, creating growth opportunity.
- Stable 20% margin reflects controlled cost structure.
